# Backtesting
En este proyecto se obtuvieron datos mediante API de Binance para realizar un backtest de una estrategia con sus indicadores como Sharpe, Max Drawdown, etc.

## Objectivo
Evaluar el desempeño de la estrategia utilizando datos históricos.

## Composicion
El archivo .py cuenta con varias funciones que se utilazaron en el archivo Notebook. Para posteriormente evaluar los resultados y ser ejecutados en otro archivo en una Raspberry pi que ejecute el codigo para abir y cerrar operaciones

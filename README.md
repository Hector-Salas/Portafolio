## PROYECTOS DESTACADOS (Experiencia práctica en finanzas)
A continuación se presentan dos proyectos realizados con Python en Visual Studio Code

### Modelo de Valor Relativo (ERP + Commodities) | Python
- Diseñé un sistema diario que calcula el Equity Risk Premium (Earnings Yield vs Bono USA 10y) y el ratio Oro/S&P 500 con ventana móvil de días.
- Implementé detección de outliers mediante Mediana de Desviaciones Absolutas (MAD) para validar cotizaciones.
- Generé reportes normativos (cierre de operaciones y Markdown con señales, alertas, etc.).
- Simulé operaciones de compra/venta respetando límites (40%-60% acciones).

### Backtesting de Estrategia para Criptomonedas | Python
- Desarrollé y evalué una estrategia de trading (media móvil 200 y Bandas de Bollinger) sobre datos históricos de criptomonedas con datos obtenidos de Binance.
- Calculé métricas de riesgo-rentabilidad: Sharpe ratio, drawdown máximo, rentabilidad anualizada, Win Rate, etc.
